**JavaScript Quiz Game
Description**

This project is a simple JavaScript Quiz Game that runs completely inside the browser console. It uses an array of questions, prompt() to take input, alert() to give feedback, and displays the final score after completing the quiz. The quiz does not require any HTML or DOM manipulation.

Objectives

Create a quiz using core JavaScript concepts

Use arrays to store questions and answers

Use loops, functions, and conditionals

Collect input using prompt()

Give feedback using alert()

Normalize input using toLowerCase() and trim()

Track and display the final score

Learning Outcomes

Understand arrays and objects in JavaScript

Use for loops to iterate through data

Write reusable functions

Use prompt() and alert() for simple I/O

Apply string methods for validation

Maintain and update score variables

Improve logical thinking and problem-solving
